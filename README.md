# 👋 Bem-vindo ao meu GitHub!

Fala aí! Eu sou o Lucas, **Engenheiro de Dados** focado na construção de pipelines, automação de processos e desenvolvimento de soluções que transformam dados em ativos estratégicos.

Atuo no desenvolvimento de soluções de ingestão, transformação e orquestração de dados utilizando **Python, SQL e Airflow**, além de automações com **n8n** e aplicações internas com **Agentes de IA**. Tenho experiência na integração de sistemas, modelagem de dados e evolução de arquiteturas analíticas.

Sou movido por aprendizado contínuo e interessado em tudo que envolve **engenharia de dados, automação e IA aplicada ao desenvolvimento**.
